# Multipurpose Business Landing Page

A modern, high-performance single-page business website built for a frontend/full-stack internship assessment. The project is designed with a performance-first mindset, focusing on responsive UI, fast loading speed, SEO optimization, and clean component-based structure.

## Features

* Sticky responsive navigation with smooth scrolling
* Hero section with call-to-action
* Services and product showcase cards
* CTA section for conversions
* “Why Choose Us” section
* FAQ accordion
* Testimonials / feedback section
* Blog preview cards
* Functional contact form using EmailJS
* Embedded Google Map / placeholder
* Responsive footer with quick links and social icons

## Tech Stack

* React
* Vite
* Bootstrap 5
* Custom CSS
* EmailJS

## Performance & SEO

This project is optimized to meet modern web standards:

* WebP image optimization
* Lazy loading for below-the-fold images
* Mobile-first responsive design
* Lighthouse score target: 90+ Performance and SEO
* JSON-LD structured data for FAQ, Articles, and Breadcrumbs
* robots.txt and sitemap.xml included
* Minified CSS and JavaScript through Vite build
* Optimized for Core Web Vitals:

  * Fast Largest Contentful Paint (LCP)
  * Minimal Cumulative Layout Shift (CLS)
  * Low First Input Delay (FID)
